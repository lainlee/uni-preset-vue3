# @lainlee/uni-preset-vue3

# What's this

This is a template for uni-app
The preset packages is:

- vue3
- typescript
- vite
- vueuse - v9
- pinia - v2
- luxon
- tailwindcss
- lodash-es
- sass

# Usage

```
npx degit lainlee/uni-preset-vue3 my-vue3-project
```
